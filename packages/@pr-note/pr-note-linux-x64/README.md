# [@pr-note/pr-note-linux-x64](https://www.npmjs.com/package/@pr-note/pr-note-linux-x64)

This is the Linux X64 (Intel) build of [@pr-note/pr-note](https://www.npmjs.com/package/@pr-note/pr-note), a tool to generate or update GitHub Pull Request (PR) notes summarizing unmerged PRs between two branches.
