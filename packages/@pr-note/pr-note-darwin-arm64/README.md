# [@pr-note/pr-note-darwin-arm64](https://www.npmjs.com/package/@pr-note/pr-note-darwin-arm64)

This is the Darwin ARM64 (Apple Silicon) build of [@pr-note/pr-note](https://www.npmjs.com/package/@pr-note/pr-note), a tool to generate or update GitHub Pull Request (PR) notes summarizing unmerged PRs between two branches.
