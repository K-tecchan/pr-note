# [@pr-note/pr-note-win32-arm64](https://www.npmjs.com/package/@pr-note/pr-note-win32-arm64)

This is the Windows ARM64 build of [@pr-note/pr-note](https://www.npmjs.com/package/@pr-note/pr-note), a tool to generate or update GitHub Pull Request (PR) notes summarizing unmerged PRs between two branches.
